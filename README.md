# CV Master - Resume Builder App

![React](https://img.shields.io/badge/React-18-blue)
![Tailwind CSS](https://img.shields.io/badge/Tailwind-3-38bdf8)
![Firebase](https://img.shields.io/badge/Firebase-10-ffca28)
![Vite](https://img.shields.io/badge/Vite-5-646cff)

A modern, professional resume builder application that helps job seekers create stunning resumes in minutes. Built with React, Tailwind CSS, and Firebase.

## ✨ Features

- **3 Professional Templates**: Choose from Elegant, Modern, and Creative designs
- **Live Preview**: See your resume update in real-time as you edit
- **PDF Export**: Download your resume in professional PDF format
- **Cloud Storage**: Your data is safely stored in Firebase
- **Easy Editing**: Simple form-based interface, no design skills needed
- **Secure Authentication**: Google Sign-in and email/password authentication

## 🚀 Tech Stack

- **Frontend**: React 18, Tailwind CSS, Vite
- **Backend**: Firebase (Authentication, Firestore)
- **Animations**: GSAP, Lucide Icons
- **Build Tool**: Vite

## 🛠️ Installation

1. Clone the repository:
```
bash
git clone <repository-url>
```

2. Install dependencies:
```
bash
npm install
```

3. Set up Firebase:
   - Create a Firebase project at [firebase.google.com](https://firebase.google.com)
   - Enable Authentication (Google & Email/Password)
   - Enable Firestore Database
   - Copy your Firebase config to `src/firebase.js`

4. Run the development server:
```
bash
npm run dev
```

5. Build for production:
```
bash
npm run build
```
